# Agent Packages

This directory holds packaged agents for the AgentHub MVP.

Current contents:

- `_template/`: starter files for new packages
- `legal-checker/`: scaffold for the LLM-only demo agent
- `clause-extractor/`: scaffold for the tool-enabled demo agent

The contract for every package is documented in [docs/AGENT_PACKAGE_CONTRACT.md](/home/lore/workspace/agenthub/docs/AGENT_PACKAGE_CONTRACT.md).
